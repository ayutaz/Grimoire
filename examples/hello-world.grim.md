# Hello World Example

## 魔法陣の説明

最も基本的なGrimoireプログラムです。

## 構造

```
    ◎
    |
    ☆
    |
"Hello World"
```

## 実行結果

```
Hello World
```

## 解説

1. `◎` (二重円): プログラムのエントリーポイント
2. `|` (接続線): 実行フローを示す
3. `☆` (5点星): 標準出力への出力操作
4. `"Hello World"`: 出力する文字列リテラル

## 描き方のポイント

- 二重円は他の円より少し大きく描く
- 接続線は垂直にまっすぐ引く
- 文字列は引用符で囲むことを忘れずに
