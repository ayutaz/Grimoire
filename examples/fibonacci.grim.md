# フィボナッチ数列の例

## 魔法陣

```
○ fib(#n)
├─ △ (n <= 1)
│  ├─true─→ ☆ n
│  └─false─┐
│          ↓
│    □ #a = fib(n-1)
│          |
│    □ #b = fib(n-2)
│          |
│    □ #sum = a + b
│          |
└─────────→☆ sum

◎ main
├─ □ #input = 10
├─ □ #result = fib(input)
└─ ☆ "fib(10) = " + result
```

## 解説

### 関数定義
- `○ fib(#n)`: 整数パラメータnを受け取るfib関数
- 関数本体は円に接続された要素

### 条件分岐
- `△`: n <= 1の条件判定
- true分岐: nをそのまま返す
- false分岐: 再帰計算を実行

### 再帰呼び出し
- `fib(n-1)`と`fib(n-2)`を再帰的に呼び出し
- 結果を加算して返す

## より視覚的な表現

```
        ○ fib
       /│\
      / │ \
     △  │  #n
    /\  │
   /  \ │
  ☆    \│

         │

    │         │

         ☆
```

## 実行結果

```
fib(10) = 55
```
