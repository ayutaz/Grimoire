# フィボナッチ数列の例

## 魔法陣

```
○
├→ □•
│
├─ △
│  │╲
│  ≤• ╲>•
│  │   ╲
│  ☆    │
│       │
│   ┌───┴───┐
│   │       │
│   ○-•     ○-••
│   │       │
│   └───┬───┘
│       │
│       +
│       │
└───────☆

◎
├─ □• = ⦿
├─ ○
└─ ☆
```

## 解説

### 関数定義
- `○`: フィボナッチ関数
- `□•`: 整数型パラメータ
- 関数本体は円に接続された要素

### 条件分岐
- `△`: 条件判定
- `≤•`: 1以下かチェック
- 左分岐（≤•）: パラメータをそのまま返す
- 右分岐（>•）: 再帰計算を実行

### 再帰呼び出し
- `○-•`: n-1での再帰呼び出し
- `○-••`: n-2での再帰呼び出し
- `+`: 結果を加算
- `☆`: 結果を出力

### メインプログラム
- `◎`: メインエントリ
- `□• = ⦿`: 入力値10を設定
- `○`: フィボナッチ関数を呼び出し
- `☆`: 結果を表示
