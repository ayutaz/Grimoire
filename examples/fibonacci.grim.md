# フィボナッチ数列の例

## 魔法陣

```
     ╱───────────────╲
   ╱        ○         ╲   ← 術式の容器
  │    ├→ □•         │   ← 入力値n
  │         │         │
  │         △         │   ← n ≤ 1の判定
  │        ╱│╲        │
  │      ≤• │ >•      │
  │      │  │  │      │
  │      ☆  │  │      │   ← nをそのまま返す
  │         │  │      │
  │    ┌────┴──┴───┐  │
  │    │           │  │
  │    ○-•      ○-•• │   ← 再帰的な螺旋
  │    │           │  │
  │    └─────┬─────┘  │
  │          │        │
  │          ⟐        │   ← 結合の印
  │          │        │
  │          ☆        │   ← 結果の具現化
   ╲                 ╱
     ╲─────────────╱

     ╱───────────╲
   ╱      ◎      ╲      ← メインエントリ
  │  ├─ □•=⦿    │      ← 入力値10
  │  ├─ ○        │      ← 術式を発動
  │  └─ ☆        │      ← 結果を具現化
   ╲            ╱
     ╲────────╱
```

## 解説

### 関数定義
- `○`: フィボナッチ関数
- `□•`: 整数型パラメータ
- 関数本体は円に接続された要素

### 条件分岐
- `△`: 条件判定
- `≤•`: 1以下かチェック
- 左分岐（≤•）: パラメータをそのまま返す
- 右分岐（>•）: 再帰計算を実行

### 再帰呼び出し
- `○-•`: n-1での再帰呼び出し
- `○-••`: n-2での再帰呼び出し
- `⟐`: 結合の印（加算）- 2つのエネルギーを1つに収束
- `☆`: 結果を出力

### メインプログラム
- `◎`: メインエントリ
- `□• = ⦿`: 入力値10を設定
- `○`: フィボナッチ関数を呼び出し
- `☆`: 結果を表示

## 魔法陣の原理

フィボナッチ数列は自然界の螺旋パターンを表現する神聖な数列です。この魔法陣では：

1. **外周円**: すべてのエネルギーを封じ込める境界
2. **再帰的螺旋**: 関数が自己を呼び出す無限の渦
3. **結合の印（⟐）**: 2つの過去の値が1つの未来を生む
4. **エネルギーの流れ**: 上から下へ、分岐から収束へ

フィボナッチ数列の10番目の値（55）が表示されます。
