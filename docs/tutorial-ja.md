# Grimoireチュートリアル

## はじめに

Grimoireへようこそ！このチュートリアルでは、文字を使わずに純粋な記号と図形だけでプログラムを作成する方法を学びます。

## 最初のプログラム: 星を表示

### 1. 基本的な出力

最も簡単なGrimoireプログラムは以下のような魔法陣です：

```
    ◎
    │
    ☆
```

- `◎`: メインエントリーポイント（二重円）
- `│`: 実行フロー
- `☆`: 出力/表示（5点星）

このプログラムは画面に星を表示します。

### 2. 画像として描く

1. 白い紙またはデジタルキャンバスを用意
2. 黒いペンまたはブラシを使用
3. はっきりとした線で図形を描く
4. 図形間の接続線を忘れずに

## 変数と計算

### 変数の宣言とデータ型

```
    ◎
    │
    □• = •⁵     ← 整数型変数に5を代入
    │
    □•• = ⦿⦿•••  ← 浮動小数点型に23を代入
    │
    □◐          ← ブール型（true）
    │
    ☆
```

データ型は図形内のパターンで識別：
- `•`: 整数型（単一の点）
- `••`: 浮動小数点型（二重点）
- `≡`: 文字列型（三本線）
- `◐`: ブール型（半円）
- `※`: 配列型（星形配置の点）
- `⊞`: マップ型（グリッドパターン）

### 数値の表現

```
• = 1
•• = 2
••• = 3
•⁴ = 4
•⁵ = 5
⦿ = 10
⦿• = 11
⦿⦿ = 20
⊡ = 100
⊙ = 1000
```

### 算術演算

```
    □• = ⦿     ← 10を代入
      ╲
       +        ← 加算
      ╱
    □• = •⁵    ← 5を代入
      │
    □•         ← 結果（15）
      │
      ☆
```

演算子記号：
- `+`: 加算
- `-`: 減算
- `×`: 乗算
- `÷`: 除算

## 条件分岐

### 基本的な条件分岐

```
    □• = •⁵    ← 変数に5を代入
    │
    △          ← 条件判定
   ╱ ╲
  <⦿  ≥⦿      ← 10より小/以上
 ╱     ╲
☀       ☾      ← 開始/終了
```

比較演算子：
- `<`: より小さい
- `>`: より大きい
- `≤`: 以下
- `≥`: 以上
- `=`: 等しい
- `≠`: 等しくない

## 関数

### 関数定義

```
○            ← 関数定義
├→ □•       ← パラメータ1（整数）
├→ □•       ← パラメータ2（整数）
│
□• = +       ← 加算処理
│
←□•         ← 結果を返す
```

### 関数呼び出し

```
◎
├─ □• = •••  ← 3を代入
├─ □• = •⁴   ← 4を代入
├─ ○         ← 関数呼び出し
└─ ☆         ← 結果を表示
```

## 配列とコレクション

### 配列の作成と操作

```
□※           ← 配列型変数
├─ •         ← 要素1
├─ ••        ← 要素2
└─ •••       ← 要素3
│
☆            ← 配列を表示
```

### マップ/辞書

```
□⊞           ← マップ型変数
├─ [•] → [⦿]    ← キー1 → 値10
└─ [••] → [⦿⦿]  ← キー2 → 値20
```

## エラー処理

### Try-Catchパターン

```
○            ← tryブロック
│
～～～       ← 例外可能なフロー
│
⚡           ← エラー発生
│
○            ← catchブロック
│
✗            ← 失敗を表示
```

## 並列処理

### 基本的な並列実行

```
    ⬢        ← 並列処理開始
   ╱│╲
  ╱ │ ╲
 ○  ○  ○    ← 並列タスク
 │  │  │
 ☆  ♪  ✉    ← 各タスクの出力
  ╲ │ ╱
   ╲│╱
    ⬢        ← 同期/結合
    │
    ✓        ← 成功
```

### カウンタープログラム

```
◎
├─ □• = ∅    ← カウンタを0で初期化
│
└─ ⬟ ← ⊡    ← 100回ループ
   │
   □• = +•   ← カウンタに1を加算
   │

   │╲
   │ ☆       ← 10の倍数なら表示
   │
   ⟲         ← ループ継続
```

## 描画のコツ

### 1. 明確な図形
- 図形の境界をはっきりと描く
- 円は完全に閉じる
- 角がある図形は角をしっかり

### 2. 接続線
- 線は図形の境界に確実に接触
- フローの方向を明確に示す
- 必要に応じて矢印を使用

### 3. パターンと記号
- データ型パターンを正確に描く
- 演算子記号を明確に配置
- 数値は点の配置で表現

### 4. 空間の使い方
- 関連する要素は近くに配置
- 複雑なプログラムは領域を分けて整理
- 余白を適切に使う

## デバッグ

### デバッグシンボル

```
○            ← 関数
├─ 👁        ← ウォッチポイント
├─ 🐛        ← ブレークポイント
└─ 📍        ← アサーション
```

## まとめ

Grimoireでは、文字を使わずに純粋な記号でプログラムを表現します：
- すべての概念が視覚的記号で表現される
- プログラムフローが空間的に理解できる
- 数値も記号の組み合わせで表現

さあ、あなたも記号だけで魔法陣を描いてプログラミングを始めましょう！
